Our Portfolio

School Management Website for Tenderminds School

A comprehensive web platform developed for Tenderminds School to manage student records, staff profiles, results, and financial transactions. The website features a user-friendly interface for teachers to upload results, allowing students to access their performance and download reports in PDF format. The admin panel provides full control over student data, financial records, and user management. Built with PHP, MySQL, and integrated with Paystack for school fees payments.

Technologies Used: PHP, MySQL, Paystack, HTML, CSS, JavaScript

JAMB CBT Practice App

A mobile and web-based Computer-Based Test (CBT) practice app designed to help students prepare for JAMB exams. It allows students to take simulated tests based on past questions, with features like a timer, automatic scoring, and detailed reports. Admins can upload new questions and options, and the app shuffles questions for each user to provide a unique test experience.

Technologies Used: PHP, CSS, JavaScript, HTML

Report Card Generator System

An automated report card generator system that simplifies the process of generating and downloading student report cards. The system supports various formats for academic data input and produces customizable report cards for each student. Teachers can easily upload results and generate PDF reports for students. Developed using PHP, MySQL, and JavaScript for interactive user interfaces.

Technologies Used: PHP, MySQL, JavaScript
