# FemTech Portfolio - Olawuni Babafemi David

Personal portfolio website for Olawuni Babafemi David, founder of FemTech - a web development brand based in Mowe, Ogun State, Nigeria.

## Features
- VSCode-inspired dark theme
- Code rain background animation
- Typing effect introduction
- SVG logo animations (bounce/wiggle on hover)
- Smooth section transitions
- Project cards with fade-in animations
- Mobile responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript
- PHP (contact form ready)

## Colors
- Primary Blue: #00BFFF
- Background: #1e1e1e (VSCode Dark)
- Secondary: #252526
- Text: #d4d4d4

## Setup Instructions
1. Replace placeholder images in `/assets/images/`
2. Add your project screenshots
3. Update social media links in contact section
4. Add your actual project URLs in project cards
5. Customize the bio text if needed
